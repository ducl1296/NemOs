NemOs


NemOs es una distribucion basada en debian, preconfigurada para verse hermosa(en mi perspectiva)
ademas de funcional, sin software de mas, simplemente lo necesario
y una que otra funcion mamalona (increible)
NemOs quiere ser hermosa, libre y rapida, sin software espia.
Aun asi se encuentra en desarollo puede tener errores y puede que algunas cosas no funcionen bien.
Si quieres puedes colaborar conmigo o solo notificar el error si te pones en contacto via telegram o correo electronico!
Tan rapido como pueda o disponga de tiempo me gustaria solucionarlo.

Libre como un delfin

    Como instalar Sistema en disco

    Requisitos

  • Usb mayor 4gb
  • Iso de la version que gustes
  • Win32DiskImager
    1. Descargar e instalar Win32DiskImager Puedes usar Rufus o Etcher, la guia se basara en Win32DiskImager.
    2. Abre Win32DiskImager
    3. Abrimos el selector

    4. Cambiamos la seleccion de imagen por defecto a *

    5. Seleccionamos la iso a grabar en la memoria

    6. Seleccionamos la memoria

    7. Presionamos sobre Write

    8. Confirmamos que YES!!!

      Precaucion, presta atencion porque se borrara toda la informacion del dispositivo que elegiste


    9. Esperamos a que cargue

    10. Esta listo!!!

    11. Inicia desde la usb, seleccionando el arranque desde la bios

    12. Datos para iniciar la live cd



      User: nemo
      Password: facedown

    1. Una vez booteada la imagen, lucira asi!

    2. Presionamos Tecla Windows + D y nos habre el selector de aplicaciones en donde buscamos Systemback

    3. Nos abrira el programa y se vera asi!
      Seleccionamos Instalar sistema

    4. Ingresaremos a una interfaz como esta, en la que llenaremos con nuestros datos!
      Es recomendable poner el nombre de usuario como "nemo" para no desconfigurar
      Todo lo demas se puede modificar sin problemas
      Una vez lleno se da click a siguiente

    5. Se abrira la configuracion para la particion de disco duro, Deberas tener cuidado ya que puedes borrar contenido!
      Seleccionas la raiz del disco duro, le das eliminar
    6. Seleccionas la particion seleccionada, seleccionas el espacio que deseas asignar y clickeas la palomita!

    7. Seleccionamos el punto de montaje el cual por defeto es /
      Seleccionamos Transferir archivos y datos de configuracion de usuario
      Y click en la flecha.

    8. Saldra este cartel y clickeamos iniciar

    9. Comenzara a cargar y lucira como la imagen, posterior a ello saldra un mensaje que notifica que se ha instalado correctamente.


    10. Reinicia y a disfrutar tu nuevo sistema recien instalado!
